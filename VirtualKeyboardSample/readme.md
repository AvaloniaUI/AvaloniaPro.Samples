# Virtual Keyboard Sample

This sample demonstrates the Avalonia Virtual Keyboard control with RIME Chinese input method support.

## Features

- On-screen keyboard with `VirtualKeyboardScope` for automatic show/hide
- Multiple input methods: English (US), German, and Chinese (RIME)
- Various text input fields (plain text, email, multiline)

## Requirements

- An Avalonia Pro or higher license key that includes `Avalonia.Controls.Keyboard`
- .NET 10.0 or newer

## Running

```bash
dotnet run
```

Tap on any text field to bring up the virtual keyboard. Use the language switch button on the keyboard to cycle between input methods.

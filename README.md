# Accelerate.Samples
Sample projects for Avalonia Accelerate UI Components

To set this up, you must have your Avalonia Accelerate credentials ready and set the `AVALONIA_LICENCE_KEY` environment variable to your license key, or explicitly set `AvaloniaUILicenseKey` in the `Directory.Build.props` file. 

Your license key can be found in your the Avalonia portal and will be of the format `avln_on_key:v1:` or `avln_off_key:v1:` followed by a string of characters.

If you have not yet obtained a license key, you can sign up for a free trial on our [portal](https://portal.avaloniaui.net/avalonia).

For additional details, see:
- [MediaPlayer Quick Start](https://docs.avaloniaui.net/accelerate/components/media-player/quickstart)
- [WebView Quick Start](https://docs.avaloniaui.net/accelerate/components/webview/quickstart)
- [TreeDataGrid Quick Start](https://docs.avaloniaui.net/accelerate/components/treedatagrid/quickstart)

Projects in this repository also have Developer Tools configured. See [Developer Tools Getting Started](https://docs.avaloniaui.net/accelerate/tools/dev-tools/getting-started) to install the tool and learn how to use it.
